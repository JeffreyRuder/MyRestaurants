# EatAt

## Description

This is an Android app created as practice in several important Android concepts, such as intents, viewpagers and viewholders, recyclerviews, and saving user information and preferences in a database.

## Features

* Gets restaurant data from the Yelp API.
* Users can login and save restaurants they are interested in.
* Users can swipe to delete restaurants from their lists of saved restaurants, and there are several other animations incorporated into the program.

## Technologies

Android, Java, XML.

## Setup

Ensure you have Android Studio installed.

Clone this repository into your AndroidStudioProjects directory.

Use AVD Manager to run the project on a virtual device, or connect an Android phone with USB debugging enabled to run on a real device.
